## Algorithm
### 1. 题目
```
70. 爬楼梯
```
### 2. 题目描述
```
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

示例 1：

输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
示例 2：

输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶
```

### 3. 解答：
```golang
func climbStairs(n int) int {
	var dp = make([]int, n+1)
	dp[0], dp[1] = 1, 1
	for i := 2; i <= n; i++ {
		dp[i] = dp[i-1] + dp[i-2]
	}
	return dp[n]
}
```
### 4. 说明
* 定义dp存储到达数字所需要的步骤
* `dp[0]`, `dp[1]` 都为1
* 状态转移方程为 f(x)=f(x−1)+f(x−2)

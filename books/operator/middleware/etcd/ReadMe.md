# etcd运维手册
## 安装etcd
1. 下载安装包
```
https://github.com/etcd-io/etcd/releases
```
2. 启动
```
nohup ./etcd &
```
3. 文档
[https://etcd.io/docs/](https://etcd.io/docs/)

## 安装etcdkeeper
1. 下载安装包
```
https://github.com/evildecay/etcdkeeper/releases
```
2. 启动
```
nohup ./etcdkeeper &
```
3. 文档
[https://github.com/evildecay/etcdkeeper](https://github.com/evildecay/etcdkeeper)
